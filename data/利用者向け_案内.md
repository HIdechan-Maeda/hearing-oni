# 聴覚・音響の鬼 — 利用者向け案内（まとめ）

授業・配布用に、この1ファイルに利用手順をまとめています。自由にコピー・改変して構いません。URLの例は運用に合わせて差し替えてください。

---

## このアプリの開き方（基本）

- **インターネット経由**で公開されたURLから利用します。自宅のWi‑Fiでも、モバイル回線（4G/5G）でも構いません。
- **PCと同じWi‑Fiにいる必要はありません。** 担当教員が案内する **本番のURL**（例: Vercel で公開された `https://◯◯◯.vercel.app` など）をスマホのブラウザで開いてください。
- **App Store / Google Play からのインストールはありません。** よく使う場合は、下の「ホーム画面に追加」でアイコンを置くと、ほかのアプリと同じように起動しやすくなります。

担当教員から共有された **アプリのURL** をブックマークしておくと便利です。  
（実際の文字列は授業の案内に従ってください。）

---

## 1. ブラウザでアプリを開く

- **スマホ・タブレット・PC** いずれでも利用できます。
- **推奨ブラウザ**
  - **iPhone・iPad**: **Safari**
  - **Android**: **Chrome**
- 上記のURLをアドレス欄に入力するか、授業資料のリンクをタップして開いてください。

---

## 2. 新規登録（はじめて使う方）

1. 画面に **メールアドレス** と **パスワード** を入力します。
2. **「新規登録」** ボタンを押します。
3. **利用できるメールアドレス**
   - **学内メール**（例: `@hoku-iryo-u.ac.jp`）は、そのまま新規登録できます。
   - **学外メール**を使う場合は、**担当教員から許可されたアドレスだけ**登録できます。許可がない場合は教員に問い合わせてください。
4. 画面に **仮登録完了**の案内が出たら、大学のメールボックスを開きます。
5. **届いたメール内のリンク**をタップ（またはクリック）して、登録を完了してください。  
   - メールが見当たらない場合は、**迷惑メールフォルダ**も確認してください。
6. 登録完了後、もう一度アプリのURLを開き、同じメールとパスワードで **「ログイン」** してください。

※ パスワードは、他人に教えず、自分だけが分かるものを設定してください。

### 新規登録できないとき

| 画面・状況 | 考えられる原因と対処 |
|------------|------------------------|
| 学内メールではない／許可されていない | **学外メール**は、担当教員が **許可リストに追加するまで** 新規登録できません。教員にメールアドレスを伝え、許可後にもう一度試す。 |
| 「既に登録されています」 | そのメールでは **以前に登録済み** です。**ログイン**を試す。仮登録メールだけ届いていない場合は「確認メールを再送」を試す。 |
| パスワード関連のメッセージ | **短すぎるパスワード**は弾かれることがあります。より長いパスワードにする。 |
| **email rate limit exceeded** など | 確認メールの送信が **短時間に多すぎる** と止められます。**30分〜1時間ほど待つ**。新規登録・再送ボタンを連打しない。 |
| それ以外の英文 | **表示された全文**をメモして教員に伝える。 |

---

## 3. プロフィール登録（学生の方・初回必須）

ログイン直後、**ニックネーム・所属・学年** の入力を求められたら、すべて入力して **「登録して学習を始める」** を押してください。  
（成績・学習状況の管理のためです。）

- **所属**で「その他」を選んだ場合は、続けて所属名を入力します。
- あとから変更する場合も、トップ画面の入力欄から更新できます。

---

## 4. スマホのホーム画面に追加する（任意）

よく使う場合は、**アプリのようにホーム画面から開ける**ようにしておくと便利です。  
※ これは **Webサイトをショートカットとして置く** 操作です。ストアからインストールするアプリとは別です。

### iPhone・iPad（Safari）

1. **Safari** でアプリのURLを開いた状態にします。
2. 画面下の **共有ボタン**（□に矢印のアイコン）をタップします。
3. 下の方にスクロールして **「ホーム画面に追加」** をタップします。
4. 名前を確認して **「追加」** をタップします。

### Android（Chrome）

1. **Chrome** でアプリのURLを開きます。
2. 右上の **︙（メニュー）** をタップします。
3. **「ホーム画面に追加」** または **「インストール」** をタップします（表示は機種やChromeの版で少し違うことがあります）。
4. 案内に従って追加します。

## 5. うまくいかないとき

| 状況 | 試すこと |
|------|----------|
| メールのリンクを押すと別の画面になる | 担当教員に **正しいアプリのURL** を確認する。メールは **古い案内のリンク** の可能性があります。 |
| 新規登録できない | 学内メールか、教員から許可されたメールか確認する。 |
| **ログインできない** | 下のチェックリストを順に確認する。 |

### ログインできないときのチェックリスト

1. **メールの「本登録」が済んでいるか**  
   新規登録直後は「仮登録」です。届いたメール内のリンクを開いてからでないとログインできません。
2. **メールアドレスの打ち間違い**  
   学内メールのスペル（`@` の前後など）を確認する。
3. **パスワード**  
   大文字・小文字の違い、入力ミスがないか確認する。忘れた場合は担当教員に相談する（再設定の手順は運用に依存します）。
4. **同じブラウザで開いているか**  
   別の端末やシークレットモードで、ブックマークした古いURLを開いていないか確認する。
5. **何度も試さない**  
   短時間に何度も失敗すると一時的にロックされることがあります。しばらく時間をおいてから試す。

### 「登録されている」のにログインできない

管理画面や教員側で「ユーザーがいる」と見えても、**メールの本登録（確認）が終わっていない**とログインできないことがあります。新規登録直後は **仮登録** であり、**届いたメール内のリンクを開いたあと**に初めてログインできます。

- メールが見つからない → 迷惑メールを確認する。
- メールが消えた・期限切れ → アプリのログイン画面にある **「確認メールを再送する」** で、同じメールアドレス宛に再送できる。

それでも解決しない場合は、**画面に表示されるメッセージ全文**をメモして、担当教員に相談してください。

## 付録: 開発・動作確認用（ローカル開発）

以下は **開発者がPCでサーバーを動かしているとき** に、スマホで画面を確認するための方法です。**一般の利用者向けの手順ではありません。**

### 同じ Wi‑Fi（LAN）でアクセス

1. **PC でサーバーを起動**
   ```bash
   npm run dev:lan
   ```
2. **ターミナルに表示された Network の URL を確認**
   - 例: `Network: http://192.168.11.7:3000`
   - この **192.168.x.x** が PC の IP です
3. **スマホを同じ Wi‑Fi に接続**し、ブラウザで `http://192.168.x.x:3000` を開く（ポートは表示に合わせる）

**つながらない場合の確認**

- **Mac のファイアウォール**: システム設定 → ネットワーク → ファイアウォール → オプションで「Node」または「node」の受信を許可
- **PC の IP 確認**: システム設定 → Wi‑Fi → 詳細 → TCP/IP の「IP アドレス」
- スマホが **同じ Wi‑Fi**（ゲストネットではない）か確認

### トンネルでアクセス（別ネットでも可）

同じ Wi‑Fi でもつながらない場合、**ngrok** や **localtunnel** で一時的な URL を発行できます。

#### localtunnel（インストール不要で試せる）

1. PC で `npm run dev` または `npm run dev:lan` を起動したままにする（例: ポート 3000）
2. **別のターミナル**で:
   ```bash
   npx localtunnel --port 3000
   ```
3. 表示された URL（例: `https://xxx.loca.lt`）をスマホのブラウザで開く
4. 初回だけ「Click to Continue」などのボタンを押すことがあります

#### ngrok（登録が必要だが安定）

1. https://ngrok.com でアカウント作成
2. インストール: `brew install ngrok` または https://ngrok.com/download
3. PC で `npm run dev` を起動したまま、別ターミナルで:
   ```bash
   ngrok http 3000
   ```
4. 表示された `https://xxxx.ngrok.io` をスマホで開く
